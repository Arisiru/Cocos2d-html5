Class: GridLayout

cc.plugin.layout. GridLayout

A grid layout lays elements out either in rows or columns. If rows are specified, the lay out will keep the fixed number of rows and grow on the number of columns or vice versa, like as follows: 

    3 rows                        3 columns

    +------------+-----...        +----------+----------+----------+
    |  row1      |                |   col1   |   col2   |   col3   |
    +------------+-----...        +----------+----------+----------+
    |  row2      |                |          |          |          |
    +------------+-----...        .          .          .          .
    |  row3      |                .          .          .          .
    +------------+-----...
