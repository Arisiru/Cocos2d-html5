Class: SpriteFont

SpriteFont

A sprite font is a fast drawing type of font that for webgl autobatches with very good performance results. This kind of fonts are backed by a texture, where the characters are already drawn. The font basically blits characters to the screen. Currently, a SpriteFont can be created:
  • From a glyph designer output file
  • A FNT output file, like the ones coming from https://www.glyphite.com
  • From Texture packer JSON output Fonts can be cached in the AssetManager.
