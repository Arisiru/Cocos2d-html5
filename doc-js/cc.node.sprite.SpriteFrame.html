Class: SpriteFrame

cc.node.sprite. SpriteFrame

This Object defines a pixels source (image, canvas, texture, etc.) and an associated Rect for image blitting operations. It has parent-child capabilities. A SpriteFrame can create subFrames. SubFrames will have a parent reference, share the same Texture instance and will apply the appropriate offset.
