Class: PropertyInfo

cc.action. PropertyInfo

Internal helper Object to store a property information. It stores:
  • a property name, for example 'x' or 'p0.x'
  • the property path. For example ['x'] or ['p0','x']
  • original property value for a target object It is responsible for setting and getting the deep property path values too. Referenced properties MUST be numeric.
