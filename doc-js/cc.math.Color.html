Class: Color

cc.math. Color

A color is represented by 4 components: RGBA encapsulated in a Float32Array.
Internally, Color components are stored as normalized color values 0..1
This object has cache capabilities for internal color string representation so calling repeatedly getFillStyle, getHexRGB and getHexRGBA will always be fast.
